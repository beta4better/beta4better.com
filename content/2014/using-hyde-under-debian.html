--- title: 在 Debian 下配置 Hyde 运行环境 date: 2014-03-19 14:00:00 tags: ['hyde','debian','python'] --- [Hyde][hyde]，[之前写过一篇](/2013/hello-hyde.html)提到了她，网站切换到 [Hyde][hyde] 的时候。 以前我并不是很注重文档这种东西，大约是要装什么的时候用搜索引擎来处理，搜到结果，问题解决就好了。 有想过用 WIKI 之类的东西来记录，但是都没有坚持下来。 但是过了一段时候后发现又会遇到同样的问题，还得用搜索引擎来处理，很是浪费时间。 所有如果时间允许的话，还是整理一下比较好。 下面便是在 Debian 下配置 Hyde 环境的一些基本方法，比较简单，解决问题就好。 关于安装，可以选择全局安装，也可以创建独立的虚拟环境。 后者比较好，方便后期的管理，万一出了什么问题，直接把这个目录删掉就好了， 不会影响到系统中的其他依赖关系。 首先，安装必须的软件包： sudo apt-get install python-dev build-essential python-virtualenv 其次，创建独立的虚拟环境： virtualenv --no-site-packages hyde-env 激活虚拟环境，如果需要退出，直接运行 `deactive` 命令即可： source www.beta4better.me/bin/activate 安装 hyde： pip install hyde 然后就可以正常使用了。 大致是这样，比较简单。 [hyde]: http://github.com/hyde/hyde [bootstrap]: http://github.com/twitter/bootstrap
