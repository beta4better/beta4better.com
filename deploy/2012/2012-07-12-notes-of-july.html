天气：晴

前言

这几日以来的第一个晴天吧，青岛的雨，要么许久不下，要下的时候会持续一整天的下。昨天晚上算是个例外，雷阵雨，还挺大。

Abby 同学的工作算是又有着落了。从 H 公司离开去卖轮胎，再到现在去卖猫食。虽然有些坎坷，但是终还是一种经历，比我这样毕业后就一直在一家公司呆着要好，至少见识了各式各样的公司，也接触了各种各样的人。

昨天公司办公室正式布置，晚上过去小帮了一下。办公室很明亮，也很气派。晚上的聚餐也很和睦，像是一家人。我觉得 Abby 同学会发大财的，哈哈……

上周五证领了之后去办了贷款，正好赶上利息下调，还算是蛮顺利的。要等一个周左右的时间才能去签贷款合同，这段时间算是可以好好休息了。

周末天气不好，没有出门，在家琢磨了 Obtvse 好久，不过最终还是没能配置好，拖到了昨天晚上，竟然奇迹般的成功了。RVM 的网站变化挺大，安装方式也跟之前不一样了，于是把之前安装的 RVM 删掉重新装了一次，之后通过 Passenger 重新安装Nginx，竟然奇迹般的把 Obtvse 运行了起来，

什么是 Obtvse ？

Obtvse 使用 ruby 实现的，另外我还找到了3个 python 的版本，样子长的都差不多，功能也类似。黑白风，简洁版的 blog 程序，可以让人专注于写作。跟 jekyll 和 octpress 推崇的理念差不多。

关于博客

我一直计划这要写一篇关于博客的文章，说说自己的一些想法。

所谓博客，写者也。以十指之韵，抒胸中之志。

  1. 为什么要写博客？

每个人对这个问题的回答都会不一样，在我自己看来，博客也是一种积累。每每看到别人的博客归档里有四、五年的存档，很是羡慕。我大概也是在四、五年前开始断断续续的写过一些，不过因为当时折腾的成分比较多，数据都没能够留存下来。不仅仅是服务器上的数据，有很多本地的数据也都没有留下来多少。少年轻狂吧。不过这些数据，只有到丢了之后才会想到重要性。

记得许久之前，有一个叫做“阳光艾芙”的博客，博主经常会写一些游览的文字，还配有一些拍的很不错的图片，经常去看。主题用的是 wordpress 的默认主题，只不过修改背景和链接的颜色。不过现在也已经不能访问了。

  1. 博客程序

上面提到的 wordpress 便是很有名的一款博客程序：成熟，用户量大，但是也渐显臃肿了。只要服务器支持 PHP 就可以运行，而且有各种各样的主题和插件。同时 wordpress.com 提供有免费和收费的博客托管服务。

Wordpress 算是动态博客中比较有名的，比较有代表性。类似的还有 typecho ， habari ，textpattern 等。大致这么多吧。像 drupal 应该不能归类到博客程序中来，drupal 这类的更侧重于是 CMS 。

另一种算是半动态半静态的，比如说 movable type 。脚本语言是 perl ， 有开源版本，也有商业版本。后台写完文章之后会生成静态的 HTML 文件。单就浏览来看能够在一定成都上减轻服务器的负担。不过每写一篇文章都有可能会将整个网站 rebuild 一次，这样的话太频繁的更新对服务器的负载也不小。
